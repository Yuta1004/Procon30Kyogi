# エラーコード対応表

## GETALLBATTLE001

### 内容

参加している試合の情報取得中、サーバから取得したJSONのデコードに失敗しました。

### 対処法

システムは時間をおいてサーバへの再接続を試みます。  
エラーが続く場合はゲームサーバ運営者に問い合わせて下さい。
